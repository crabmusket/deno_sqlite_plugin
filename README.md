# deno_sqlite_plugin

Bindings to [rusqlite](https://github.com/jgallagher/rusqlite) for [deno](https://deno.land).

## Stability

**UNSTABLE**

This plugin will panic if anything goes slightly wrong.
Probably don't use it in production just yet.
